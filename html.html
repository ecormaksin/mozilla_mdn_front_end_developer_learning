CSSとJavaScriptの適用

リンク先

CSS

<link rel="stylesheet" href="{CSSファイルのパス}" />

JavaScript

<script src="{JavaScriptファイルのパス}" defer></script>

"defer" はHTMLのパースが終わった後でJavaScriptをロードする指定。HTMLのパースが終わる前にJavaScriptがロードされ、存在しない要素に対して処理が行われるとエラーになってしまうので、それを防ぐために指定する。

強調と重要度

リンク先

強調

<p>I am <em>glad</em> you weren't <em>late</em>.</p>

イタリック体にするために使ってはいけない。
イタリック体にする場合は、<span>とCSSを使うか、<i>を使う。

重要な単語の強調

<p>This liquid is <strong>highly toxic</strong>.</p>

ボールド体にするために使ってはいけない。
ボールド体にする場合は、<span>とCSSを使うか、<b>を使う。

強調を入れ子にもできる。

<p>This liquid is <strong>highly toxic</strong> — if you drink it, <strong>you may <em>die</em></strong>.</p>

ブロッククォート

  <p>
    The <strong>HTML <code><blockquote></code> Element</strong> (or
    <em>HTML Block Quotation Element</em>) indicates that the enclosed text is
    an extended quotation.
  </p>
</blockquote>

インラインクォート

<p>
  The quote element — <code><q></code> — is
  <q cite="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/q">
    intended for short quotations that don't require paragraph breaks.
  </q>
</p>

出典

<cite> タグは出典元のタイトルをマークアップする場合などに使う。このタグ自体はそれ以外の機能を持っていないので、<a> タグで囲むことで出典元へリンクできる。

  <a href="/en-US/docs/Web/HTML/Element/blockquote">
    <cite>MDN blockquote page</cite></a>:
</p>

<blockquote

  <p>
    The <strong>HTML <code><blockquote></code> Element</strong> (or
